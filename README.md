# Movie_Rating_Analysis

This project focuses on analyzing movie ratings to understand audience preferences, rating trends, and factors influencing movie success. Using Python and data analysis techniques, the project explores how ratings vary across different movies and identifies key patterns in user feedback.

Tools & Technologies :

Python

Pandas

NumPy

Matplotlib

Seaborn

Jupyter Notebook

Techniques Used :

Data Cleaning and Preprocessing

Exploratory Data Analysis (EDA)

Statistical Analysis

Data Visualization

Trend and Pattern Analysis

Key Analysis & Insights :

Identified movies with the highest and lowest ratings

Analyzed rating distribution across movies

Observed trends in audience preferences

Compared average ratings and vote counts
